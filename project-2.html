Project Overview

IncidentTrack is a full-stack incident reporting system built to facilitate seamless communication between security officers and property managers. Officers can create and update detailed reports, which are automatically summarized using AI and sent to property managers via email. Property managers can filter, view, and manage these reports by unit number, date, or resolution status, eliminating paperwork and improving efficiency.
